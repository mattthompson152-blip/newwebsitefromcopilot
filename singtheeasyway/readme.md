## 📌 Overview

This repository contains the complete source code for the **Sing The Easy Way** website — a modern, fully responsive multi‑page site designed for online singing lessons.

The website includes:

- Light/Dark mode toggle  
- Neon accent UI  
- Clear structured content  
- Six fully designed pages  
- Floating CTA buttons  
- Mobile‑friendly layout  
- Safe placeholder areas for images  
- Booking + enquiry buttons (SimplyBook & JotForm ready)
